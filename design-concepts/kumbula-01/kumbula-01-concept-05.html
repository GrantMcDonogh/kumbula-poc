Step 03

Connect Repository

KumbulaCloud uses a push-to-deploy model. Add the git remote below, then push your branch. Every push to main triggers a new build automatically.

Terminal — Git Setup
$ git remote add kumbula git@deploy.kumbula.cloud:aurora-api.git
$ git push kumbula main
# Alternatively: push any branch for a preview deployment
$ git push kumbula feature/my-branch
SSH key required. Make sure your public key is added under Account → SSH Keys before your first push. Builds stream live once received.
